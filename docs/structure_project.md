# Project's structure

```txt
visual_slam_project/
│
├── config/                      # Конфигурации сенсоров, модели, параметры фильтров
│
├── data/                        # Видеофайлы, датасеты, результаты логов
│
├── src/
│   ├── core/                    # Ядро системы (общая логика, графы, трансформации)
│   │   ├── tracker.py           # Базовая логика трекинга и локализации
│   │   ├── frame.py             # Структура одного кадра / наблюдения
│   │   ├── map.py               # Представление карты (локальной или глобальной)
│   │   └── pose_graph.py        # Фактор-граф, хранение связей и оптимизация
│
│   ├── sensors/                 # Работа с источниками данных
│   │   ├── camera.py            # Камеры: захват, undistortion, FPS
│   │   ├── stereo.py            # Работа с синхронной стерео-парой
│   │   ├── rgbd.py              # Работа с RGB-D сенсорами (RealSense и др.)
│   │   └── imu.py               # Поддержка IMU (на перспективу)
│
│   ├── vision/                  # Обработка изображений и 3D
│   │   ├── depth_estimation.py  # Получение глубины (stereo, mono, RGB-D)
│   │   ├── pointcloud.py        # Построение и фильтрация облаков
│   │   ├── features.py          # Извлечение и сопоставление фичей (ORB, SIFT и др.)
│   │   ├── triangulation.py     # Триангуляция и восстановление 3D-точек
│   │   └── alignment.py         # ICP, трансформации, совмещение облаков
│
│   ├── perception/              # Анализ сцены, семантика, препятствия
│   │   ├── occupancy.py         # Карта проходимости, voxel grid
│   │   ├── segmentation.py      # Семантический или геометрический разбор сцены
│   │   └── environment.py       # Обработка локального окружения (например, вокруг точки A)
│
│   ├── navigation/              # Планирование движения (может быть stub на MVP)
│   │   ├── planner.py           # Алгоритмы (A*, DWA, RRT и др.)
│   │   └── controller.py        # Интерфейс с исполнительным уровнем
│
│   ├── api/                     # Интерфейс для управления пайплайном
│   │   └── server.py
│
│   ├── utils/                   # Вспомогательные модули
│   │   ├── visualization.py     # Open3D, matplotlib и др.
│   │   ├── geometry.py          # Матрицы, преобразования, векторы
│   │   ├── filters.py           # Пространственные и статистические фильтры
│   │   └── log.py               # Логирование, профилирование
│
│   └── main.py                  # Главная точка входа — запуск пайплайна
│
├── notebooks/                   # Исследования, тестирование частей пайплайна
│
├── tests/                       # Тесты для модулей (vision, sensors и т.п.)
│
├── requirements.txt
├── README.md
└── .gitignore
```
